# Социальная сеть "Mesto"

Проект сервиса, где можно делиться своими путешествиями.
Реализовано открытие попапа, добавление карточек и редактирование профиля. Страница полностью адаптивна. Разработка велась согласно принципам ООП.

### Стек технологий

JavaScript, HTML5, CSS3. 

### Ссылки на проект

https://darvinnn.github.io/mesto/

### Как клонировать и запустить проект

```bash
# Клонировать репозиторий
git clone https://github.com/darvinnn/mesto.git

# Перейти в папку с проектом
cd mesto

# Установить зависимости
npm install

# Запустить проект в режиме разработки 
npm run dev
```

Откройте браузер и перейдите по адресу `http://localhost:3000`, чтобы начать работу с платформой.
