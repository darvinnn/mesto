# Проект: Mesto

**О проекте**

Проект сервиса, где можно делиться своими путешествиями. Работа выполнена в рамках практики по модулю с введением в JavaScript на Яндекс.Практикуме.
Реализовано открытие попапа, редактирование профиля. Страница полностью адаптивна. Файловая структура организована согласно БЭМ nested.

*Ссылка на страницу*
https://darvinnn.github.io/mesto/
